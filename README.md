# PycukSystems

Базовая подготовка репозитория к разработке системы.

## Что добавлено
- Документ проектирования (`docs/design.md`).
- Скрипт инициализации окружения (`scripts/setup_env.sh`).
- Базовый каркас структуры проекта для дальнейшей разработки.

## Быстрый старт

```bash
bash scripts/setup_env.sh
source .venv/bin/activate
```

После этого можно устанавливать зависимости проекта и начинать разработку.

## Рекомендуемая структура

- `docs/` — документация и дизайн-схемы.
- `scripts/` — служебные скрипты подготовки/автоматизации.
- `src/` — исходный код системы (создаётся по мере разработки).
- `tests/` — автотесты.
