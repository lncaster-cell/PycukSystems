# NWN2 stock scripts archive

Этот каталог содержит эталонный (upstream) набор скриптов NWN2, который используется как **read-only архив include-файлов** при компиляции.

## Источник
- Базовые скрипты из дистрибутива Neverwinter Nights 2 / официальных кампаний и контента Obsidian.
- Файлы перенесены в репозиторий без функциональных модификаций проекта.

## Назначение
- Предоставлять стандартные include-зависимости и reference-реализации для локальной компиляции.
- Служить эталоном при сравнении нашего кода с оригинальными реализациями.

## Политика обновления
- Каталог считается внешним слоем (`compilator`).
- Ручные правки отдельных `.nss` в этом каталоге не допускаются.
- Обновление выполняется только пакетной синхронизацией из источника с явным описанием версии/набора в сообщении коммита.
